# Tamil Songs Downloader

A simple command-line tool for downloading Tamil MP3 songs organized by categories like Star Hits, Music Directors, Singers, and more.

## Features

- **Multiple Categories**: Browse songs by Star, Music Director, Singer, Old Hits, and Ringtones
- **Batch Downloads**: Download entire albums or individual songs
- **High Quality**: 320kbps MP3 audio support
- **Organized Output**: Songs automatically sorted into folders by category and album
- **Easy to Use**: Interactive menu-driven interface.

## What's Included

- Interactive CLI with simple navigation
- Download options for:
  - Star Hits (Rajinikanth, Kamal Haasan, Vijay, Suriya, etc.)
  - Music Director Hits (A.R. Rahman, Ilayaraja, Yuvan Shankar Raja)
  - Singer Collections (Anirudh, Harris Jayaraj, and more)
  - Old Tamil Songs (70s, 80s, 90s)
  - Ringtones and Instrumental tracks
- Real-time download progress tracking
- Smart folder management

## Installation

### For Windows Users

Download the latest `tamil-mp3-downloader-v2.0.0.exe` from the [Releases](https://github.com/anburocky3/tamil-mp3-downloader/releases) page and run it directly. No installation needed.

### For Developers

**Requirements**: Python 3.10 or higher
```bash
# Clone the repository
git clone https://github.com/anburocky3/tamil-mp3-downloader.git
cd tamil-mp3-downloader

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

## How to Use

1. Run the application
2. Select a category from the main menu
3. Choose your preferred artist, director, or collection
4. Select specific albums or download all
5. Wait for downloads to complete
6. Find your songs in the `output` folder, organized by category and album

## Output Structure

Downloaded songs are saved in:
```
output/
  ├── StarHits/
  ├── MusicDirectorHits/
  ├── SingerHits/
  ├── OldSongs/
  └── Ringtones/
```
